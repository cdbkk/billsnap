# BillSnap Email Templates

Email templates for Supabase authentication.

## Setup Instructions

1. Go to Supabase Dashboard: https://supabase.com/dashboard/project/wyvybpodjkksvmmmmexl/auth/templates

2. For each template below, click on the template name, paste the HTML, set the subject, and save.

## Templates

| File | Supabase Template | Subject |
|------|-------------------|---------|
| `confirm-signup.html` | Confirm signup | `Verify your BillSnap account ✓` |
| `reset-password.html` | Reset Password | `Reset your BillSnap password` |
| `magic-link.html` | Magic Link | `Sign in to BillSnap` |
| `invite-user.html` | Invite User | `You're invited to BillSnap` |

## Important Settings

Make sure email confirmations are enabled:
1. Go to Authentication → Providers → Email
2. Enable "Confirm email" toggle

## Preview

Open any `.html` file in a browser to preview how it looks.
