# Receipt Generation & Sharing System

Complete Thai thermal receipt system for BillSnap mobile app.

## Overview

This system allows users to:
1. Generate professional Thai-style thermal receipts
2. Preview receipts before sharing
3. Share receipts via iOS/Android share sheet
4. Save receipts to device gallery

## Files Created

### Core Components

1. **/components/ReceiptPreview.tsx**
   - Renders authentic Thai thermal receipt
   - Supports `forwardRef` for ViewShot capture
   - Features: monospace fonts, centered header, right-aligned prices
   - Shows watermark for free tier users

2. **/components/ReceiptModal.tsx**
   - Full-screen modal with receipt preview
   - Share and Save to Gallery buttons
   - Loading states during operations
   - Handles permissions automatically

3. **/lib/share.ts**
   - `captureReceipt(ref)` - Capture receipt as PNG image
   - `shareReceipt(uri)` - Share via native share sheet
   - `saveToGallery(uri)` - Save to device gallery
   - `captureAndShare(ref)` - Combined capture + share
   - `captureAndSave(ref)` - Combined capture + save

### Documentation

4. **/components/README.md** - Component usage guide
5. **/components/INTEGRATION_EXAMPLE.tsx** - Full working example
6. **/docs/RECEIPT_SYSTEM.md** - This file

## Quick Start

### Basic Usage

```typescript
import { useState } from 'react';
import ReceiptModal from './components/ReceiptModal';
import { calculateReceiptTotals, generateReceiptNumber } from './lib/receipt';

function MyScreen() {
  const [showReceipt, setShowReceipt] = useState(false);

  const shop = {
    id: '1',
    name: 'My Shop',
    contact: 'Tel: 02-123-4567',
    promptpay_id: '0812345678',
    is_pro: false,
    receipts_this_month: 5,
    created_at: new Date().toISOString(),
  };

  const items = [
    { id: '1', name: 'Coffee', price: 50, quantity: 2 },
    { id: '2', name: 'Cake', price: 80, quantity: 1 },
  ];

  const { subtotal, vat, total } = calculateReceiptTotals(items, true);

  return (
    <>
      <Button onPress={() => setShowReceipt(true)}>Show Receipt</Button>

      <ReceiptModal
        visible={showReceipt}
        onClose={() => setShowReceipt(false)}
        shop={shop}
        receipt={{
          items,
          subtotal,
          vat,
          total,
          receiptNumber: generateReceiptNumber(5),
          createdAt: new Date(),
        }}
      />
    </>
  );
}
```

## Integration with Supabase

When saving receipts to Supabase:

```typescript
import { supabase } from './lib/supabase';
import { generateReceiptNumber } from './lib/receipt';

async function saveReceipt(shop, items, totals) {
  // 1. Get current receipt count
  const { data: shopData } = await supabase
    .from('shops')
    .select('receipts_this_month')
    .eq('id', shop.id)
    .single();

  // 2. Generate receipt number
  const receiptNumber = generateReceiptNumber(shopData.receipts_this_month);

  // 3. Save receipt
  const { data, error } = await supabase
    .from('receipts')
    .insert({
      shop_id: shop.id,
      receipt_number: receiptNumber,
      items,
      subtotal: totals.subtotal,
      vat: totals.vat,
      total: totals.total,
    })
    .select()
    .single();

  // 4. Update shop receipt count
  await supabase
    .from('shops')
    .update({ receipts_this_month: shopData.receipts_this_month + 1 })
    .eq('id', shop.id);

  return data;
}
```

## Permissions

The app automatically handles permissions:

### Share Permission
- **Required:** None
- **Platform:** Built-in share sheet (iOS/Android)
- **Behavior:** Works immediately, no permission needed

### Save to Gallery Permission
- **Required:** `MediaLibrary` access
- **Platform:** iOS and Android
- **Behavior:**
  - First time: Shows permission dialog
  - User denies: Shows alert asking to grant permission
  - User grants: Saves image immediately
