# Receipt Components

Thai thermal receipt generation and sharing system for BillSnap.

## Components

### ReceiptPreview

A component that renders a Thai-style thermal receipt preview.

**Props:**
- `shop: Shop` - Shop information
- `receipt` - Receipt data with items, totals, and receipt number

**Features:**
- Thai thermal receipt styling (monospace, centered header, right-aligned prices)
- Automatic VAT 7% display (if applicable)
- PromptPay ID display (if set)
- Free tier watermark ("Generated by BillSnap")
- Uses `forwardRef` for ViewShot capture

### ReceiptModal

A modal that displays the receipt preview with share and save actions.

**Props:**
- `visible: boolean` - Modal visibility
- `onClose: () => void` - Close handler
- `shop: Shop` - Shop information
- `receipt` - Receipt data

**Features:**
- Full-screen modal with receipt preview
- Share button (share to other apps)
- Save to Gallery button
- Loading states during capture/share operations

## Usage Example

```typescript
import React, { useState } from 'react';
import { View, Button } from 'react-native';
import ReceiptModal from './components/ReceiptModal';
import { Shop, ReceiptItem } from './types';
import { calculateReceiptTotals, generateReceiptNumber } from './lib/receipt';

function ReceiptScreen() {
  const [showModal, setShowModal] = useState(false);

  // Your shop data
  const shop: Shop = {
    id: '1',
    name: 'My Shop',
    contact: 'Tel: 02-123-4567',
    promptpay_id: '0812345678',
    is_pro: false,
    receipts_this_month: 5,
    created_at: new Date().toISOString(),
  };

  // Your receipt items
  const items: ReceiptItem[] = [
    { id: '1', name: 'Coffee', price: 50, quantity: 2 },
    { id: '2', name: 'Cake', price: 80, quantity: 1 },
  ];

  // Calculate totals
  const { subtotal, vat, total } = calculateReceiptTotals(items, true);

  const receipt = {
    items,
    subtotal,
    vat,
    total,
    receiptNumber: generateReceiptNumber(5), // Pass existing count
    createdAt: new Date(),
  };

  return (
    <View>
      <Button title="Show Receipt" onPress={() => setShowModal(true)} />

      <ReceiptModal
        visible={showModal}
        onClose={() => setShowModal(false)}
        shop={shop}
        receipt={receipt}
      />
    </View>
  );
}
```

## Direct Share/Save (without Modal)

If you want to share or save without showing the modal:

```typescript
import React, { useRef } from 'react';
import { View, Button } from 'react-native';
import ReceiptPreview from './components/ReceiptPreview';
import { captureAndShare, captureAndSave } from './lib/share';

function DirectShareExample() {
  const receiptRef = useRef<View>(null);

  const handleShare = async () => {
    if (receiptRef.current) {
      await captureAndShare(receiptRef.current);
    }
  };

  const handleSave = async () => {
    if (receiptRef.current) {
      await captureAndSave(receiptRef.current);
    }
  };

  return (
    <View>
      {/* Hidden receipt for capture */}
      <View style={{ position: 'absolute', left: -9999 }}>
        <ReceiptPreview ref={receiptRef} shop={shop} receipt={receipt} />
      </View>

      <Button title="Share Receipt" onPress={handleShare} />
      <Button title="Save to Gallery" onPress={handleSave} />
    </View>
  );
}
```

## Permissions

The app automatically requests permissions when needed:

- **Share:** No permissions required (built-in iOS/Android share sheet)
- **Save to Gallery:** Requests `MediaLibrary` permissions on first use

## Free Tier Watermark

The watermark "Generated by BillSnap" appears at the bottom of receipts when:
- `shop.is_pro === false`

To remove the watermark, set `shop.is_pro = true` (Pro tier).

## Styling

The receipt uses authentic Thai thermal receipt styling:
- Monospace font family
- Centered shop name and header
- Dashed separators (`--------------------------------`)
- Right-aligned prices
- Item format: `qty x name ... price`
- Receipt paper color (`#FFFEF7`)

All styling can be customized in `components/ReceiptPreview.tsx`.
